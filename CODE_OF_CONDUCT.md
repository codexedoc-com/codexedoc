# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, respectful, and harassment-free experience for everyone.

## Expected Behavior

* Be respectful.
* Assume good intentions.
* Welcome newcomers.
* Provide constructive feedback.
* Respect differing viewpoints.

## Unacceptable Behavior

* Harassment
* Personal attacks
* Discrimination
* Trolling
* Spam
* Doxxing

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors whose behavior violates this Code of Conduct.

Repeated violations may result in permanent bans from the project.

## Reporting

Please report concerns privately to the project maintainers.
